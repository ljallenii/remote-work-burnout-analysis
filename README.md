# Work From Home Burnout Analysis
## Project Overview
This project explores burnout among remote workers during work-from-home (WFH) arrangements. Using survey data collected from https://www.kaggle.com/datasets/sonalshinde123/work-from-home-employee-burnout-dataset, the analysis investigates factors contributing to burnout, such as:

* Daily work hours
* Breaks taken
* Sleep hours
* Task completion rate

The goal is to uncover trends and correlations that help organizations and individuals better understand and manage WFH burnout.

## Key Features
* *Exploratory Data Analysis*: Understand the distribution of working hours, stress levels, and burnout scores.
* *Correlation Analysis*: Identify factors most strongly linked to burnout.
* *Sentiment Analysis*: Analyze free-text survey responses to detect common feelings or complaints (if applicable).
* *Data Visualization*: Includes heatmaps, bar charts, line plots, and word clouds for clear insights.
* *Actionable Insights*: Recommendations for managing burnout based on the findings.

## Technology/Libraries Used
* Python 3
* Pandas, NumPy - Data Processing
* Matplotlib, Seaborn - Visualizations
* Jupyter Notebook - Analysis workflow
